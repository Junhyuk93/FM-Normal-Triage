# FM-Normal-Triage
Normal chest X-ray triage model for fast, rule-based normal/abnormal screening in clinical workflows.

## Overview
FM-Normal-Triage is a lightweight triage model designed to quickly classify chest X-ray studies as normal or abnormal to support prioritization in radiology workflows.
